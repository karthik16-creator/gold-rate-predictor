# 🥇 Gold Rate Predictor

Predicts next-day gold rate using Machine Learning.

## Results
- Best Model : Linear Regression
- R² Score   : 0.985
- MAE        : ₹50.28/gram
- Data       : 503 days (yfinance API)

## Tech Stack
Python | Pandas | Scikit-learn | Matplotlib | yfinance

## How to Run
pip install yfinance scikit-learn matplotlib seaborn
python gold_rate_predictor.py
